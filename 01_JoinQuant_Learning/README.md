## 🎓 验收标准

完成第一阶段后，您应该能够：

1. **熟练使用聚宽平台**
   - 创建和运行策略
   - 查看回测结果
   - 理解各项指标含义

2. **掌握核心API**
   - get_price() 获取价格数据
   - get_fundamentals() 获取财务数据  
   - order_target() 下单交易
   - attribute_history() 获取历史数据

3. **实现基础策略**
   - 双均线策略年化收益率 > 10%
   - 最大回撤 < 20%
   - 夏普比率 > 1.0

4. **建立选股系统**
   - 能够筛选出50-100只优质股票
   - 选股逻辑清晰可解释
   - 回测验证选股有效性
